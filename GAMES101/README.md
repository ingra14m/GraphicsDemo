## Dependencies

- eigen

  - 通过homebrew安装

  ```
  brew install eigen
  ```

  - 默认的安装路径为`/opt/homebrew/Cellar/eigen/`，我们需要将eigen的include目录加入到cmake的编译路径中即可

- opencv

  - 有很多种安装途径
  - 需要在cmake中加入include的路径，同时链接上opencv的动态库
